Для просмотра презентации используйте стрелки навигации, либо щелкните на ней один раз левой кнопкой мыши и листайте стрелками на клавиатуре.

Как смотреть на весь экран

Презентацию можно открыть в отдельном окне или вкладке браузера. Для этого щелкните по ней правой кнопкой мыши и выберите в контекстном меню соответствующую команду. Чтобы развернуть презентацию на весь экран, нажмите на клавиатуре клавишу F.
